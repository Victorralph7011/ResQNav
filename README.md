# 🚨 ResQNav - Rescue Navigation System

**ResQNav** is a smart rescue navigation system designed to help first responders and rescue teams navigate efficiently during emergencies.

## 🚀 Features

- **Real-Time Tracking** – GPS-powered live tracking of rescue teams
- **Smart Route Planning** – AI-optimized routes considering hazards and conditions
- **Hazard Mapping** – Live hazard zone overlays with satellite data
- **Offline Mode** – Full functionality without internet connectivity
- **Team Coordination** – Integrated command center for team management
- **Analytics Dashboard** – Post-mission analytics and reporting

## 🛠️ Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Victorralph7011/ResQNav.git
   ```
2. Open `index.html` in your browser, or use a local dev server.
